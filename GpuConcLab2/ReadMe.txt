    GPU Cocurrency lab #2 .
========================================================================

This project consist of several algorithms of matrix transpose implemented with AMP C++
1) Transpose with blocks
2) Transpose with blocks without shared memory
3) Transpose with blocks with shared memory 
4) Matrix multiplication with shared mamory
5) Matrix multiplication with threads that calculates one whole row.

Text of lab: 
Реализовать следующие алгоритмы:

1) Блочная версия транспонирования.

2) Блочная версия матричного умножения без использования разделяемой памяти

3) Блочная версия матричного умножения с использованием разделяемой памяти

4) Матричное умножения с "укрупненной" декомпозицией (каждый поток вычисляет все элементы одной строки)
